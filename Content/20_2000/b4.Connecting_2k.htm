Connecting to the RAN

Integris 2000 is designed to exercise the radio head along with its associated cabling and connectors, so must be connected in place of a BBU.

NOTE: you MUST install exactly the same SFPs into the Integris test port as used in RRH & BBU, and connect using an identical cable type.

 

Information: Integris ports use Small Form Factor Pluggable (SFP) devices to provide signal conversion of the optical signal on the cable to the electrical signals used inside the Integris system. SFP's are available for many types of physical connection, such as single mode or multi-mode optical cabling, and the specification of SFP's is controlled by a standard which ensures compatibility between SFP's from different manufacturers. 

Part of this standard ensures that all vendors provide machine readable information regarding the SFP's capabilities. This information is called the Vital Product Data (VPD) and includes the SFP manufacturer, supported link rates, optical wavelength, etc. 

The Integris System can read this information and present it to the user so they can verify that the SFP is suitable for the application in which it is being used.

Radio Head Installation Testing & Monitoring

To test or monitor a radio head installation connect the Integris system in place of the BBU/eNB as described below. (See Fig 1.2 )

  1. If the radio head is already cabled to a BBU/eNB, locate the cable going to the radio head to be tested, then disconnect it from the SFP in the BBU/eNB (See Fig 1.3 )

    Information: If you have any doubt that you have the correct SFP in the Integris 2000 test port, remove the SFP from the BBU/eNB port and use that. This will ensure the correct specification SFP is used for testing

     

    Removing the cable from the the BBU/eNB

  2. Plug the cable from the radio head into an Integris test port (See Fig 1.4 )

    Integris 2000 Test Ports (port 1 at left)
  3. Run the Install test sequence (see the section on Installation Testing)
