The Integris 2000 Install tests consist of a sequence that initializes the link and Remote Radio Head (RRH), then verifies that the link is capable of reliably supporting CPRI traffic at the current speed.

Note: Radio Head testing includes checking for a response from the radio head unit. If a simple loop-back is connected then the Radio Head tests will detect this and fail the test (shown as a red 'X').

If you want to self-test the Integris system with a loop-back connector then select the Selftest configuration file before testing.

 

Information: For some manufacturer's equipment, Install tests return inventory and metrics like VSWR, Transmit power and RSSI. They can also include enhanced inventory listing and firmware levels.

 

Note: Before testing you will need the following information:

  • The correct configuration file for the manufacturer and the communications standard being used (e.g. Lucent LTE)
  • The CPRI Link Rate, sometimes referred to as Link Speed

The Status Pane to the left of the screen expands during Installation tests to give you more control options. (See Fig 1.1 )

Link Status: displays the current link Status

Link Rate: Opens a drop-down list for link rate selection (See "Setting the Link Rate", page 1)

(See "Setting the Link Rate", page 1)

Max AxC: This setting sets the maximum number of antennas that will be polled as part of the installation test.

When set to Auto Integris 2000 polls all available antennas and reports their data. You can also limit the number of antennas that Integris 2000 polls by setting a value between 1 and 4. This can be useful, for example, when an antenna is temporarily removed or shut down.

Freq Disp: Allows you to choose between displaying the channel number or frequency display for the transmitter and receiver

Tx Freq/Chan: The current transmit frequency or channel as selected above. This is set by the configuration file.

Rx Freq/Chan: The current receiver frequency or channel as selected above. This is set by the configuration file.

Tx Power: Displays the transmitter power output

BW: Displays the current signl bandwidth. This is set by the configuration file.

OCNS: Applies Orthogonal Channel Noise Simulation (OCNS) to simulate user load on the downlink

PCI: Allows you to set the Physical Cell identification for the cell
