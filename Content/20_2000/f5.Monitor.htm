Monitor Mode

Location  
GUI or Keyboard CTRL + M
Classic menu Test > Monitor
Modern menu Test > Monitor

Integris 2000 tests are set to run until the end of the sequence, then conclude with a Pass or Fail result.

Radio head monitoring is a long term test designed to exercise the radio head for as long as required, usually to enable crews to adjust or modify the installation.

Note: it is also possible to run monitoring tests to the local SFP. Although monitoring of the whole installation is more common, SFP monitoring could be used to detect intermittent faults.

To monitor radio head operation:

  1. Select the Test menu, and click on New.
  2. Click on the Monitor menu option (Optionally click on CTRL + M). If using Classic menus, the text under the MONITOR button changes to a green ON.

  3. Click Run (Optionally press F5).

    The word "Monitoring" will appear in yellow at the top right of the window to indicate status. The test results pane shows the current values for the test.

  4. To stop monitoring click on Monitor again.

Note: Unlike the other tests, monitoring will continue until you click on the Stop button or disable monitoring. If you stop then restart monitoring, all values will be reset.
