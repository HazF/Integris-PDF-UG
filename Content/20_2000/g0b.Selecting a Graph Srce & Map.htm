Selecting a Graph Source & Mapping

Location  
GUI or Keyboard Source icon at top left of the screen
Classic menu Analyze > Graph > Source
Modern menu Graph > Source

 

  
GUI or Keyboard Mapping icon at top left of the screen
Classic menu Analyze > Graph > Mapping
Modern menu Graph > Mapping

Note: The following must be set before any of the graph types can be displayed

Before you can display a graph you must first decide whether the source data will come from a live signal on one of the two test ports, or from a recorded file.

To select the Graph source:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Source.
  3. The two test ports and a list of available recorded files will appear similar to Fig 1.1

    Selecting a graph source

  4. Select the desired Graph source from the list

Once you have decided the source of the data, you must then select the file containing the mapping information to convert the signal data into a meaningful graph.

Information: Integris 2000 mapping files contain information on frequency, bandwidth and other information required to decode the graph from the CPRI stream.

To select a Mapping:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Mapping.
  3. A list of the available mapping files will appear similar to Fig 1.2

    Selecting a graph source

  4. Select the desired Graph source from the list

Once the correct source and mapping file have been selected a graph view should appear on the screen.
