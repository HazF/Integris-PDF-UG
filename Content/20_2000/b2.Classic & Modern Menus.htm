Selecting Classic or Modern Menus

Menu Location  
GUI or Keyboard None
Classic menu System > Prefs...
Modern menu System > Preferences...

Integris allows you to select between two menu structures to facilitate easier operation depending on the working environment.

  • Classic menus are better when using a touch screen
  • Modern menus are closer to standard Windows menus and may be easier to use with a keyboard and mouse.

In the sections that follow task descriptions will show the equivalent locations in both menu systems, e.g. "Classic menu Test > LoadConfig" means go to the Classic Test menu and select the Loadconfig menu item.

To swap between menu systems:

  1. Open the System menu
    • Click on Prefs on the Classic menu system, or
    • Click on Preferences on the Modern menu system
  2. From the Preferences pane click the Interface dropdown menu and select either the Classic or Modern menu system as required (See Fig 1.1 )
  3. Click Done
