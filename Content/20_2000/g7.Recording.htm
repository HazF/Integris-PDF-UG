Recording & Replaying RF Data

Menu Location  
GUI: Record icon at top right of the screen  
Classic Menu Analyze > Source > Record / Playback  
Modern Menu Source > Record / Playback  

Note: This feature is ONLY available when graph views are displayed and selected.

You can record graphs to be replayed later, Recordings are fixed duration of approximately 50 seconds, and have a file size of approximately 130 Megabytes.

Information: By default, recording are saved in the proprietary .IQS format and are saved to the C:\Integris-2000\AA_IQS_TRACES folder. You can give the files any name you desire but by default are created using the date and then the time (e.g.20151125_084539_695.iqs

To record a graph:

  1. Either

    • Click on the circular red icon on the recording control panel at the top right of the screen. (See Fig 1.1 ) A Select Destination File dialog will open to allow you to name and save the recording

      OR

    • (If using the Classic Menu system, first click to open the Analyze menu.) Open the Source menu and click on Record. A Select Destination File dialog will open to name and save the recording
  2. Click OK to start recording. A progress bar will indicate how far through the fixed length recording you are.

 

Record screen controls

To replay a graph:

  1. (If using the Classic Menu system, first click to open the Analyze menu.) Open the Source menu and click on Playback. (See Fig 1.1 ) A Select File dialog will open to allow you to select a file for replaying
  2. Click OK to playback the file.
