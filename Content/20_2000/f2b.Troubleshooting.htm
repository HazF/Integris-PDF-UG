Troubleshooting a RAN Installation

If a radio head installation test fails the initial Integris 2000 tests then there is a logical debug sequence to follow to locate the component giving the problem. This section suggests a sequence of actions and details the required tests.

Test local SFP

If the Install test fails at the physical layer phase of the test a logical start point is to test the local SFP

With the current SFP still in Integris test port 1:

  1. Fit a known good loopback cable to the SFP being tested (See Fig 1.2 ).
  2. Open the Emulate menu and select New.
  3. Now click on Run from the same menu (or press the F5 key),

Inserting an SFP for testing

Loopback Connection

If this (local) SFP passes the SFP test, remove the SFP in the radio head and insert that into the Integris test port (See Fig 1.3 ). Fit the loopback cable and repeat the test.

If this SFP passes then replace it into the radio head. You now know you have good SFPs in the Integris test port and the radio head.

Testing the Cable

The next logical step is a test of the cable linking the Integris system to the Radio Head. This procedure assumes the SFP currently in the Integris test port has passed the previous SFP test.

If the radio head is at the top of the tower, it is advisable to fit the loopback at the lowest connection in the link, then progressively work your way up the tower testing as you go.

  1. Insert one end of the cable into Integris test port 1
  2. Fit a looback to the far end of the cable (See Fig 1.2 )
    • An alternative if no loopback connector is available is to remove one of the optical fibers from the end clip that makes up the duplex cable, and insert opposite ends of the same fiber into the TX and RX connectors of Integris test port.
  3. Open the Emulate menu and select New.
  4. Now click on Run from the same menu (or press the F5 key),

Repeat the test at Ground Level

Bringing the Radio Head back to ground level makes replacing components, and unit configuration easier.

Connecting Integris 2000 to an RRH

To test a radio head installation connect the cable assembly to the Integris test port. (See Fig 1.3 ).

  1. Take a full duplex cable and connect one end to the RRH

  2. Connect the other end of the cable to the SFP in the Integris System test port. (See Fig 1.4 )(See Integris 2000 Test Port 1 ).

    Note: you MUST install exactly the same SFPs into the Integris test port as used in RRH & BBU. If there is any doubt as to the SFP's specification, remove the SFP from the BBU and use that in Integris test port 1

  3. Repeat the Integris Install test sequence by Opening the Emulate menu and selecting New.
  4. Now click on Run from the same menu (or press the F5 key),

Monitor Mode Testing

Radio head monitoring is a long term test designed to exercise the radio head over a period of time. This allows installation crews time to adjust or modify the installation.

(See also "Monitor Mode" on page 1)

(See also Monitor Mode)

To monitor radio head operation:

  1. Select the Emulate menu, and click on New.
  2. Click on the Monitor menu option (Optionally press CTRL + M). If using Classic menus, the text under the MONITOR button changes to a green ON.

  3. Click Run (Optionally press F5).

    The word "Monitoring" will appear in yellow at the top right of the window to indicate status. The test results pane shows the current values for the test.

  4. To stop monitoring, select the Test menu and click on Stop
