Real Time, Max, Min & Average Values

Displaying Real Time, Max, Min & Average values

Menu Location  
GUI or Keyboard None
Classic menu Analyze > Trace > Trace
Modern menu Trace > Trace / Enable

By default Integris 2000 graphs display a real-time spectrum. By using the Trace Control Panel you can add or remove Maximum, Minimum, Real-Time and Average values onto the existing graph.

Realtime, Max, Min and Average values

Using the Classic Menu System:

  1. Select the Analyze menu
  2. Open the Trace menu, then click on the Trace menu item. This will open a dropdown list of graph value sets as described above
  3. Click on RealTime, Maximum, Minimum or Average as desired (See Fig 1.2 )
  4. Once you have the desired graph type selected, click the Enable button so that it displays 'On'. You can open any combination, or all values on the same graph (See Fig 1.1 ).

Classic Traces menu

Using the Modern Trace Control Panel:

  1. Open the Trace menu and click on the Configure menu item. This will open the Trace Control Panel panel (See Fig 1.10 ).
  2. You now have a number of options:
    • Click on Enable to activate the desired spectrum type.
    • Click on Reset to clear and restart the spectrum.
    • Click on the respective Hold and Decay boxes to open a dropdown list that allows you to select the desired values. (Hold and Decay are only available for Maximum and Minimum graphs. For Real-Time and Average graphs they are grayed out).
  3. Click on the All Off menu item to deactivate the current graph (The Trace Control Panel remains on the screen).
