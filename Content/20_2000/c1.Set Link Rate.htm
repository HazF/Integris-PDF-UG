Before testing you must set Integris 2000 to the correct link rate for the test.

Link Rate icon

 

Link status

 

To set the link rate:

  1. Click on the Rate icon in the Status panel at top left of the main window. A drop down menu will appear. The currently selected rate is highlighted
  2. Click on the required link rate. (See Fig 1.1 )

The Link text on the icon will display the words "Active" in green if the local SFP recognizes an active CPRI link, or a red error message if a problem exists (See Table 1.1 ):

Warning Meaning
None Unable to determine Link State
Active The receiver has locked at the specified bit rate
LossOfSync The receiver cannot lock at the specified bit rate and hyperframe synchronization has been lost
LossOfSignal The optical power level received at the Integris system's SFP is below its minimum sensitivity. No optical signal can be detected.
Initializing Integris hardware is initializing
NoSFP No SFP has been detected in Integris test port 1
BadSFP The Integris system is unable to communicate with the SFP fitted in test port 1. The SFP may be faulty.
NoCable No cable has been detected attached to Integris test port 1
HwFault Integris hardware fault
