Step 1: Creating a Test Report

(See also Working with Test Reports)

Report files contain one or more sets of results from the tests you run. When you start a test a default report is created for you, but it is a good idea to create one ahead of testing so you can edit the name and add details such as location.

  1. Select the Report menu and click New. A new blank report will be created with a default format file name and the current date.

    Alternatively, click on Select to open a previously saved report.

  2. On the Report menu, click on Edit. In the report window that opens add site details and any notes you might want to include. You can also edit the file name to something more appropriate.

Step 2: Starting the Install Test sequence:

  1. Ensure that the correct link rate has been set and the Status Pane is displaying the link state as ACTIVE

    To set the link rate:

    1. Click on the Rate icon in the Status Panel at top left of the main window. A drop down menu will appear. The currently selected rate is highlighted
    2. Click on the desired link rate.

  2. Ensure you have selected the correct configuration file for the radio head under test.

    To load a configuration file:

    • Click in the Test Config icon at the top left of the test view
    • On the Classic menu system, open the Test menu and click on Loadconfig
    • On the Modern menu system, open the Test menu and click on Load Config (See Fig 1.1 )

    NOTE: Turnup tests offer the most extensive radio head testing. As well as Local & Remote SFPs, Physical & CPRI Layer information and radio head inventory they include radio head metrics like VSWR, Insertion Loss and TX / RX power.

    Manufacturers' Configuration Files
  3. Decide if you want to simulate user load using OCNS. If so:

    1. Near the bottom of the Status Pane at the top left of the screen, click OCNS

    2. Select the OCNS value required to simulate the desired circuit loading.

      OCNS setting

  4. Decide if you want the radio head to transmit a Physical Cell identification (PCI). If so click on PCI at the bottom of the Status Panel, then enter the desired value.

    PCI Setting

Step 3: Running the Installation Test

  1. Open the Emulate menu and select New option. (See Fig 1.2 and Fig 1.3 )

  2. Now click on Run from the same menu (or press the F5 key),

    By clicking on the Messages icon at the top left of the test results pane you can see information returned as the test progresses.

    Install Test Results messages

    Once the test sequence has been completed you can optionally save the test results by adding them to a report.

    To add to an existing or new report file, click on the yellow Add To Report icon at the top right of the result panel.

    (See also Working with Test Reports)

    You can either choose a name for these results, or you can accept the default name based on the date and time of the test. Once you have finished editing the name, click on OK.

    NOTE: The Remote SFP results section lists information related to the SFP fitted to the Radio Head. These results should NOT be confused with those gathered during Integris 2000 SFP Tests which refer to the Local SFP fitted to the Integris system.

Step 7: Exporting the Report File in Portable Document Format (PDF)

  1. Click on Report > Export PDF. An Explorer window will open allowing you to name and save the PDF file to a location of your choice.

  2. Click Save.
