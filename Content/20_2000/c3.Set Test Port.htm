Setting the Test Port

Location    
GUI or Keyboard BBU Port icon at the top of the Main Window
Classic menu Emulate > BBU Port
Modern menu Emulate > BBU Port

The Integris system has two test ports. The one to the left is Port 1 and the one to the right is Port 2

Inserting a cable in Integris Port 1

 

Port selection icon

You can choose the desired test port by either:

  • Clicking on the BBU Port icon at the top left of the screen, then selecting the desired port. (See Fig 1.2 )
  • If using either Classic or Modern menus, open the Emulate menu and click on BBU Port. Select the desired port from the dropdown list

Note: You can use both Ports simultaneously. You can, for example, run an installtion test on Port 1 while testing an SFP on Port 2.

When using multiple screens you can have one screen set to Port 1 and another set to Port 2. (See Fig 1.2 )
