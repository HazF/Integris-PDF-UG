Integris 2000 Graph Views

Opening a Graph View

Location  
GUI or Keyboard Not applicable
Classic menu Analyze > Graph > New
Modern menu Graph > New

In addition to the installation tests, Integris 2000 can display 'digital' spectrum graphs, including power spectrum, persistence and/or waterfall graphs derived from the IQ data from the Radio head and carried over a CPRI link.

Metrics collected from the transmitted and received signals allow the presence of Implied Passive Intermodulation distortion (PIM) to be deduced.

Opening graphs & locating on screen

To display a power graph:

  1. If using the Classic menu system, open the Analyze menu.

  2. Click on the Graph menu menu and select New. (See Fig 1.1 )

    Opening a Graph view

  3. On the Create Graph dialog box that opens:

    1. Select the graph Type(See "Selecting the Graph Type", page 1)

      (See also Selecting the Graph Type)

    2. Select the graph Source. You can select live CPRI data from Integris port 1 or 2, or playback from a file.

      3. Selecting a graph type

      Selecting the graph source

  4. Click on Ok

  5. The original graph(s) will now appear with blue positioning borders to allow you to place the new graph in relation to it. Fig 1.4 shows the left positioning bar selected which will place the new graph to the left of the current graph. Fig 1.5 shows the new graph displayed to the left of the original graph.

    New graph plus Positioning Borders

    New graph placed to the left of the original (plus view control icon at top right)

Closing Graphs

Closing Graphs

You can close the current graph by right clicking on it and selecting Close View

Alternatively open the Trace menu and click on All Off.
