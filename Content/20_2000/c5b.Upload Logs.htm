Uploading System Logs

Location  
GUI or Keyboard Not applicable
Classic menu System > Help > Upload Logs
Modern menu Help > Upload Logs

If your support contact requests that you upload the system logs, then perform the following steps.

  1. If using the Classic menu system, open the System menu.
  2. Click on the Help menu item
  3. From the Help menu that opens, click on Upload Logs
  4. After a few seconds a report screen will open confirming the upload, and giving you the report identifier. (See Fig 1.1 )

Note:Please make a note of the identifier and the time it was uploaded in case your support asks for this.
